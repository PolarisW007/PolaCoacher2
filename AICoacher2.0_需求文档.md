> 版本：2.0

---

## 一、项目概述

### 1.1 产品定位

AICoacher 2.0（品牌名：**AI 藏经阁 2.0**）是一款基于 AI 的智能文档学习与知识传播平台。Slogan 为 **"重构你的学习体验"**。

平台以 **文档** 为核心载体，提供从文档上传、AI 解析、智能讲解、语音合成、翻译对照到社交分享的全链路知识管理能力。2.0 版本在 1.0 的基础上，新增了 **多视图书架管理体系**、**文档分组与分类**、**多来源文档导入**、**在线书籍搜索导入**、**AI 对话问答**、**讲解备注面板**、**公开知识社区**、**社区评论互动** 等核心功能，实现从"知识输入 → 知识理解 → 知识管理 → 知识传播 → 社区共学"的完整闭环。

### 1.2 核心价值主张

| 维度 | 说明 |
|------|------|
| 智能讲解 | AI 驱动的文档讲解，不只是阅读，而是"听课式"深度学习 |
| 多视图管理 | 四种书架视图模式（默认/讲解/文档/分组），满足不同使用场景 |
| 全链路闭环 | 上传 → 解析 → 摘要 → 讲解 → 语音 → 翻译 → 分享，一站式完成 |
| 知识传播 | 一键生成小红书/朋友圈图文，让知识从个人走向社交 |
| 知识社区 | 讲解可公开发布至社区，用户之间互相学习、评论互动，形成知识共享生态 |
| 多格式支持 | PDF、Word、TXT、Markdown 多格式文档上传与解析 |
| 多来源导入 | 本地上传、URL 导入、文库搜索、在线书籍搜索导入，多渠道汇聚知识 |

### 1.3 技术架构

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | Python + FastAPI（异步） |
| 数据库 | PostgreSQL（asyncpg） |
| AI 引擎 | 阿里云 DashScope（Qwen Plus / CosyVoice / 通义万象） |
| 前端框架 | React 18 + Vite 5 |
| UI 组件库 | Ant Design v5 |
| 路由方案 | React Router v6（BrowserRouter，2.0 升级） |
| HTTP 通信 | Axios |
| 认证方案 | JWT（python-jose）+ OAuth2 |
| 部署环境 | 阿里云 ECS + Nginx + Supervisor |

#### 4.3.5 文档处理流程

上传/导入后自动触发处理：

```
文档入库（本地上传 / URL导入 / 文库搜索 / 书籍搜索导入）
    ↓
文档解析（PDF: pdfplumber / Word: python-docx / 网页: readability）
    ↓
逐页/逐段提取文本
    ↓
Qwen Plus 生成文档摘要
    ↓
Qwen Plus 提取关键知识点
    ↓
Qwen Plus 生成 PPT 结构化内容
    ↓
存入 PostgreSQL（实时更新处理进度）
```

## 十、核心业务流程

### 10.1 文档处理流程

```
用户上传 PDF / Word / 导入 URL
    ↓
文档解析器（pdfplumber / python-docx / readability）
    ↓
逐页/逐段提取文本
    ↓
Qwen Plus 生成摘要
    ↓
Qwen Plus 提取关键知识点
    ↓
Qwen Plus 生成 PPT 结构化内容
    ↓
存入 PostgreSQL（实时更新进度 → 前端轮询展示）
```

### 10.2 在线书籍搜索导入流程（2.0 新增）

```
用户点击"搜索书籍"入口
    ↓
输入书名 / ISBN / 作者名等关键信息
    ↓
POST /api/documents/book-search → 后端爬取 zh.annas-archive.li 搜索结果
    ↓
前端展示搜索结果卡片列表（书名、作者、ISBN、格式、大小、语言）
    ↓
用户选择目标书籍 → 点击"导入"
    ↓
GET /api/documents/check-isbn/{isbn} → 检查是否已导入
    ├── 已存在 → 提示"该书已在书架中"，终止
    └── 未导入 → 弹出确认弹窗
         ↓
    确认导入 → POST /api/documents/book-import
         ↓
    创建文档记录（status: importing）→ 返回 task_id
         ↓
    后台异步任务启动：
         │
         ├── 1. 从 Anna's Archive 下载 PDF 文件
         │      └── 实时更新下载进度 → 前端轮询 /book-import/{task_id}/status
         │
         ├── 2. 下载完成 → 文件存储到服务器
         │
         ├── 3. 自动填充书籍元信息（书名、作者、ISBN、出版社、出版年份、语言）
         │
         └── 4. 进入标准文档处理流程（解析 → 摘要 → 关键点 → PPT）
              ↓
         处理完成（status: ready）
              ↓
         发送站内通知 → "《书名》已导入成功，点击查看"
              ↓
         用户点击通知 → 跳转到文档详情/阅读页

    ──── 异常分支 ────
         │
         ├── 下载失败（书源不可用/超时）→ status: error
         │    └── 通知用户失败原因 + 提供"重试"按钮
         │         └── POST /api/documents/book-import/{task_id}/retry
         │
         └── 文件损坏/无法解析 → status: error
              └── 通知用户 + 建议手动上传
```

### 10.3 AI 讲解生成流程

```
用户点击"生成讲解"
    ↓
基于 PPT 内容 + 原文分页
    ↓
Qwen Plus 并行生成每页讲解（4 并发）
    ↓
    ├── 后台翻译（英→中，3 并发）
    └── 后台预生成音频（CosyVoice，2 并发）
    ↓
用户进入播放器查看/收听
```

### 10.4 社交分享流程

```
用户点击"生成小红书/朋友圈"
    ↓
Qwen Plus 生成标题 + 文案
    ↓
通义万象生成封面图
    ↓
返回可分享的图文内容
```

### 10.5 书架管理流程（2.0 新增）

```
用户进入书架页面
    │
    ├── 未登录 → 展示空书架框架 → 点击操作 → 弹出登录弹窗
    │                                              │
    │                                         登录成功
    │                                              │
    └── 已登录 → 加载书架数据 → 展示文档卡片列表
                     │
                     ├── Tab 切换 → 默认 / 讲解 / 文档 / 分组
                     │
                     ├── 添加文档 → 本地上传 / URL导入 / 文库搜索 / 搜索书籍导入
                     │
                     ├── 分组管理 → 创建 / 编辑 / 删除分组
                     │
                     ├── 点击文档 → 进入阅读/讲解页面
                     │
                     └── AI 问答 → 基于文档内容的智能对话
```

### 10.6 讲解备注流程（2.0 新增）

```
用户进入讲解播放器
    ↓
播放讲解内容（语音 + 文本 + PDF）
    ↓
点击右侧 ▶ 展开备注面板
    ↓
在当前页输入备注内容
    ↓
自动保存（防抖 1s → PUT /lecture-notes/{page}）
    ↓
切换页码 → 自动加载目标页备注
    ↓
    ├── 继续记录其他页备注
    └── 点击"导出备注" → 下载 Markdown 文件
```

### 10.7 讲解发布到社区流程（2.0 新增）

```
用户在播放器页面点击"发布到社区"
    ↓
弹出发布确认弹窗
    ├── 编辑标题（默认取文档标题）
    ├── 填写简介描述（可选）
    ├── 选择分类标签（技术/学术/商业/人文/其他）
    └── 预览封面图
    ↓
确认发布 → POST /api/documents/{id}/publish
    ↓
讲解状态变为 🌐公开
    ↓
出现在社区首页 → 所有用户可浏览和播放
    ↓
    └── 随时可"撤回" → POST /api/documents/{id}/unpublish → 恢复为 🔒私有
```

### 10.8 社区浏览与互动流程（2.0 新增）

```
用户/游客进入社区首页
    ↓
浏览公开讲解卡片列表（推荐/最新/热门）
    ↓
    ├── 搜索/筛选 → 按关键词或分类标签查找
    │
    └── 点击讲解卡片 → 进入讲解详情页
         │
         ├── 播放讲解（复用 DocumentPlayer）
         │    └── 展开备注面板 → 记录"我的学习笔记"
         │
         ├── 点赞 ❤️（需登录）
         │
         ├── 收藏到书架 ⭐（需登录）
         │
         └── 评论区互动
              ├── 发表评论（需登录，限 500 字）
              ├── 回复他人评论（@用户名）
              ├── 点赞评论
              └── 查看更多评论（分页加载）
    ↓
互动产生通知 → 讲解发布者/被回复者收到站内通知
    ↓
点击通知 → 跳转到对应评论位置
```


## 十二、2.0 与 1.0 差异总结

| 维度 | 1.0 | 2.0 |
|------|-----|-----|
| 首页 | 知识繁星（简单文档列表） | 书架（四视图 Tab 切换） |
| 文档管理 | 简单列表 + 搜索 | 卡片网格 + 列表 + 分组 + 批量操作 |
| 文档来源 | 仅 PDF 上传 | PDF/Word/TXT/MD 上传 + URL 导入 + 文库搜索 + 在线书籍搜索导入（Anna's Archive） |
| 登录方式 | 账号密码 + 验证码 | 新增微信扫码登录 |
| 文档阅读 | PDF 渲染 + 摘要展示 | 新增笔记标注 + AI 对话问答 |
| 讲解备注 | 无 | 播放器右侧备注面板（可收起/展开），逐页关联，自动保存，支持导出 |
| 讲解可见性 | 无（所有讲解仅自己可见） | 支持私有/公开切换，公开后发布到社区供所有用户浏览收听 |
| 知识社区 | 无 | 公开社区首页（推荐/最新/热门），支持搜索、分类筛选、收藏 |
| 社区互动 | 无 | 点赞、评论、回复、评论点赞，轻量级学习交流 |
| 站内通知 | 无 | 新评论/回复/点赞通知，未读角标，一键已读 |
| 翻译 | 讲解自动翻译 | 新增选中段落即时翻译 |
| 分组 | 无 | 支持自定义分组管理 |
| 空状态 | 无特殊设计 | 引导式空状态（虚线框 + 操作提示） |
| 数据存储 | 部分 JSON 文件 | 全部迁移至 PostgreSQL |
| AI 调用 | 无重试机制 | 增加重试与降级策略 |
| 音频缓存 | 无清理机制 | 自动过期清理 |
| 路由 | HashRouter | BrowserRouter |
